# ⭐ FullStack Rating App

A full-stack MERN application built with **React**, **Express.js**, and **PostgreSQL**. It supports **three roles** (Admin, User, Store Owner), authentication, and a full rating system for stores.

---

## 📌 Features

- 🔐 **Role-based login**
  - Admin: Manage all users & stores
  - User: View & rate stores
  - Store Owner: View & manage own store ratings
- ⭐ **Ratings System** (1-5 stars)
- ✅ Full authentication with JWT
- 📁 RESTful APIs (CRUD operations)
- 🛡️ Protected routes by user role
- 🧪 Auto-seeded test data

---

## 🛠️ Tech Stack

| Layer       | Tech                        |
|------------|-----------------------------|
| Frontend    | React, Axios, Tailwind CSS |
| Backend     | Node.js, Express.js        |
| Database    | PostgreSQL (via Sequelize) |
| Auth        | JWT, Bcrypt                 |
| Dev Tools   | Postman, GitHub, ESLint     |
